# URL Shorter

This project is a rails shorter URL.

## Dependencies

``Ruby 2.1.2``<br/>
``Rails 4.1.8``<br/>

## Testing

This project uses RSPEC as the main test framework, to run make:

`` bundle exec rspec ``

## Running

``bundle install``<br/>

``rake db:create db:migrate``<br/>
``rails s``<br/>
